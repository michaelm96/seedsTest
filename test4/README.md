# Test 4
- Anagram Test

for given array, should grouping them based on their anagram (consist of same character(s) but in different order)

> Before
```
let arr = [
    "kita",
    "atik",
    "tika",
    "aku",
    "kia",
    "makan",
    "kua",
];
```

> After
```
[ 
    ["kita", "atik", "tika"], 
    ["aku", "kua"], 
    ["kia"],
    ["makan"], 
]

```

## How to run 
```
node test4.js
```
